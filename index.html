Mis proyectos


Sistema de gestión de entregas de Medicamentos

Este sistema es un sistema de gestión de entrega de medicamentos que permite a los usuarios registrar, consultar y administrar información sobre pacientes, recetas y entregas. Facilita el control de inventarios, la validación de recetas y el seguimiento de las entregas, garantizando la correcta distribución de los medicamentos a los pacientes.

Sistema de gestión de producto

Este sistema es una aplicación web CRUD para la gestión de productos utilizando Python, Flask y SQL Server. Incluye carga masiva de productos desde Excel, generación de reportes en Excel, paginación de productos y eliminación individual de registros de carga de archivos.

Sistema de Gestión de Sueldos de los Empleados

Esta aplicación que permite la gestión de sueldos. También permite guardar resultados en un archivo de texto. Tecnologías Utilizadas: Lenguaje: C# Plataforma: .NET Framework Interfaz: Windows Forms

Calculadora

Desarrollé una calculadora interactiva que combina una interfaz moderna y funciones avanzadas. Permite realizar operaciones básicas y funciones adicionales como borrar dígitos. Destaca por su tema oscuro y funcionalidad de voz, pronunciando las operaciones al pasar el ratón y vocalizando los resultados. Utilicé HTML, CSS, y JavaScript para la lógica, con la API de Texto a Voz para la funcionalidad vocal. Este proyecto destaca mis habilidades en desarrollo web y la creación de experiencias únicas para el usuario.

Ir a mi sitio web de la calculadora



Autenticación con clave encriptada

Desarrollé una aplicación de escritorio en C# para la gestión segura de la autenticación de usuarios. Permite al administrador registrar usuarios con contraseñas cifradas en una base de datos SQL Server mediante BCrypt.Net. Destacando la seguridad, la aplicación incorpora la funcionalidad de bloqueo de cuentas tras tres intentos fallidos. Además, el administrador puede habilitar o deshabilitar usuarios según sea necesario. Facilita la actualización de la configuración empresarial asociada a cada usuario, respaldado por un diseño modular que permite futuras mejoras.

Ir a mi repositorio GitHub



Menú Dinámico Escritorio con DLL

He realizado una aplicación de escritorio en C# con un menú dinámico basado en archivos DLL, creados en otro entorno para optimizar su funcionalidad. La gestión de usuarios se abordó a través de SQL Server, asegurando un almacenamiento robusto. Destaco la versatilidad de la aplicación, diseñada con una arquitectura altamente escalable y modificable para facilitar futuros desarrollos. La carga dinámica de módulos subraya su capacidad para evolucionar eficientemente, adaptándose a las necesidades cambiantes del proyecto.

Ir a mi repositorio GitHub



App Android Mapas con Notificaciones

Desarrollé una aplicación Android utilizando Java como lenguaje principal. La aplicación integra la API de Google Maps para mostrar la ubicación en tiempo real del usuario en un mapa. Para acceder a la ubicación del usuario, se emplea el servicio de ubicación de Android. Además, se implementa Firebase Cloud Messaging (FCM) para la gestión de notificaciones push y la obtención del token de registro. En resumen, las tecnologías y lenguajes clave incluyen Java, la API de Google Maps y Firebase Cloud Messaging.
